# WebGoat Practice — System Security Portfolio

> **Disclaimer:** All activities documented in this repository were conducted exclusively within an isolated virtual machine environment as part of **41182 System Security** coursework. This is purely for educational purposes. Nothing here is intended for, or applicable to, use on live or unauthorised systems.

---

## About This Repository

This repository documents my hands-on practice with [WebGoat](https://owasp.org/www-project-webgoat/), a deliberately vulnerable web application developed by OWASP, used to learn and understand common web security vulnerabilities in a safe, controlled environment.

---

##  Environment Setup

| Tool | Details |
|------|---------|
| Virtualisation | VMware |
| OS (Guest) | Kali Linux |
| Application | WebGoat (OWASP) |
| Network | Host-only / NAT (isolated) |

---

## Learning Goals

- Develop practical understanding of the [OWASP Top 10](https://owasp.org/www-project-top-ten/) vulnerabilities
- Build skills in identifying and explaining security weaknesses
- Practise responsible documentation of security findings
- Connect theoretical course content to real-world application security
